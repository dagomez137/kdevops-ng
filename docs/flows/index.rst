=====
Flows
=====

Operator guides for the `Windmill`_ flows under ``f/``. The first page covers
what is common to every flow that drives a guest: how a flow surfaces its state,
and how to reach a guest from the command line when you need to look behind the
Windmill UI. The remaining pages are one per flow. These complement
:doc:`../concepts/flows`, which covers the naming model.
