# kdevops

Windmill flows for building Linux kernels and QEMU, standing up disposable
QEMU/systemd virtual machines, and running filesystem tests (fstests) against
them, reproducibly, from the browser or the CLI.

- Bootstrap a deployment: see [BOOTSTRAP.md](BOOTSTRAP.md)
- Design and usage docs: [docs/](docs/)
- Contributing, kernel-style via b4: see [CONTRIBUTING.md](CONTRIBUTING.md)
